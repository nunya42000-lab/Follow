Welcome to Follow Me 👋

This app is a memory assistant built to help you track complex sequence games and pattern challenges. It remembers the inputs so you don't have to.

The Core Loop
👀
Watch
See the pattern.
👇
Input
Record it instantly.
▶️
Play
Let the app recite it.
Top Menu Icons
  • ⏱️ Timer: Stopwatch controls for tracking time limits.
  • # Counter: Manual tally tracker.
  • 🎤 Mic: Toggle voice command listening.
  • 🎵 Tone Cadence: Toggle hum/whistle note input (9-Key only).
  • 🗒️ Gesture Pad: Open the touch-swipe input area.
  • 🖐️ Hand Tracking: Activate AI front-camera gesture tracking.
  • 📷 AR Mode: Turn background transparent via rear camera (needs HTTPS).
  • ⌨️ Bigger Buttons: Hide the UI and make every key perfectly square for maximum touch targets.
  • 🔄 Position Swap: Move the input keypad from the bottom of the screen to the top.
  • 🔲 Full Screen: Hide the browser's address bar and UI chrome.
  • 🙃 Upside Down: Flip the entire interface 180° (for a partner facing you).

Each icon above only appears in the header once its matching toggle is turned on in Settings → General.

Every toggle, slider, and switch in the Settings menu explained.

🎮 Game Tab ▼

Profiles: Save and load your entire configuration instantly.

Input: Choose your layout.
• 9-Key: Standard 3x3 grid.
• 12-Key: 3x4 grid for advanced setups.
• Piano: 7 natural keys, 5 sharp/flat keys.

Mode:
• Simon Says: Cumulative sequence. Adds one item per round, playing from the start.
• Unique: Random sequences. Clears completely each round.

Machines: Track up to 4 completely independent sequences simultaneously.

Length: Maximum sequence boundary per machine.

Auto-Advance (Unique): Automatically clears the sequence for the next round after playback finishes.

Practice Mode 🎓: Runs a dummy round with random numbers mimicking your current settings to test layouts.

▶️ Playback Tab ▼

Autoplay: Automatically triggers playback the moment a sequence round is completed.

Flash 👀: Visually highlights the number bubble during playback.

Audio 👂: Reads the sequence aloud using the selected Voice Settings.

Morse ✋: Triggers physical haptic vibrations mapped to dot/dash patterns for silent feedback.

Speed: Multiplier for how fast the sequence is delivered.

Pause: Injects a small or large delay between numbers.

Chunk Size & Delay: Breaks long sequences into groups (e.g., 3 numbers at a time) with a forced breath/delay between chunks.

Voice Profile: Select, create, and tune custom text-to-speech voices (Pitch, Speed, Volume).

⚙️ General Tab (Toggles) ▼

Theme 🎨: Pick a color scheme or open the Editor to create your own.

Timer ⏱️: Shows header stopwatch.
Auto Timer: Starts stopwatch on input.
Counter #: Shows tally counter.
Auto Counter: Increments on input.
Haptics 📳: Vibrate on button press.
Show Intro: Launch setup modal on boot.
Random Theme 🎲: Picks a random built-in or custom theme each time the app opens.
Boss Mode 🌑: Shake to black out screen.
BM Gestures: Touch-input while blacked out.
Wake Lock 💡: Prevents screen sleeping.
Upside Down: Flips the entire UI 180°.
Full Screen: Hides browser bars.
Eco Mode 🔋: Reduces animations/framerate.
Position Swap 🔄: Adds a header button that moves the input pad from the bottom of the screen to the top.
Bigger Buttons: Hides menus and makes each key perfectly square (as tall as it is wide) instead of a fixed height.
Touch Gesture: Shows the transparent touch gesture pad (taps/swipes drawn anywhere on screen).
Hand Gestures 🖐️: Master switch for the rear-camera hand tracking engine. Required for both per-key hand mapping and Hand Signals below.
Hand Signals: Global camera gestures for Clear (🤌), Delete (👌), Play (🤘), Stop (✊) - needs Hand Gestures on too.
AR Mode 📸: Rear-camera background. Needs HTTPS (or localhost) - camera permission can't be requested otherwise.
Voice Input 🎤: Master switch for the microphone/speech recognizer. Required for both spoken numbers and Voice Commands below.
Voice Commands: Say your trigger word then "play", "stop", "clear", or "delete" - needs Voice Input on too.
Tone Cadence Mode 🎵: Hum or whistle notes (C D E F G A B C D) to enter 1-9. 9-Key input only for now; requires microphone access.
Quick Erase: Hold backspace to rapidly delete.
AP Shortcut: Long-press Play to toggle Autoplay.
Vol. Gesture: 3-finger twist adjusts volume.
Speed Gesture: 2-finger twist adjusts speed.
Delete Gesture: 1-finger squiggle erases.
Clear Gesture: 2-finger squiggle nukes the round.
🔍 General Tab (Layout) ▼

Sequence Size: Scales the individual cards showing the remembered numbers, from 50% up to 300%.

UI Scale: Zooms the entire interface in or out.

Number Size: Adjusts the font size of the text inside the sequence cards.

AR Playback Speed: The speed at which your recorded AR background clips play back.

Resize Gesture: Maps the pinch-to-zoom gesture to scale the Global UI, the Sequence Cards, or disables it.

Voice Trigger: The specific word the app listens for before logging your spoken numbers or a command (e.g., "Set 1 2 3" or "Set Play").

🎛️ Mapping Tab ▼

Touch / Hand tabs: Switch between mapping touchscreen gestures and camera hand gestures to each key.

9-Key / 12-Key / Piano accordions: Each tab has one accordion per layout. Open one to see that layout's Quick Preset picker plus every key's individual dropdown.

Quick Presets ⚡: Bulk-assigns every key in that specific layout at once (e.g. "Finger Counts" for hand, "Taps" for touch) - individual keys can still be fine-tuned afterward. Every key is pre-assigned a sensible default the first time you open this tab.

Key Mappings: Assign a specific touch-screen gesture or hand gesture to each individual keypad value.

Tap Speed, Swipe Distance, and the gesture category filters moved to 🛠️ Developer Mode below.

🛠️ Developer Mode ▼

Opens from the wrench icon in the top-left of the Settings header. Holds the more technical, fine-tuning controls:

Tap Speed (ms): How fast you must release your finger for it to count as a tap. Higher = easier to tap, harder to swipe.

Swipe Distance (px): How far your finger must drag before it triggers a swipe. Higher = prevents accidental swipes.

Long Press Time, Tap Wiggle Room, Spatial Tap Precision, Long Swipe Distance, Multi-Finger Swipe Sync: Five more fine-tuning sliders for the touch engine - each has its own explanation directly under its slider.

Populate Hand/Touch Gesture Menus: Two separate filter groups. Hand controls which pose categories (Poses, Pinches, Finger Counts, Advanced Shapes) show up in the Hand tab's dropdowns; Touch controls which of the 13 touch categories (Taps, Swipes, Boomerangs, etc.) show up in the Touch tab's dropdowns.

Haptic Output Mapping: Customize the dot/dash vibration pattern for each of the 12 possible values, used when the Playback tab's Morse toggle is on.

🖐️ Front Camera AI Tracking

Control the app without touching the screen. Turn on "Hand Gestures" in Settings → General, then click the 🖐️ icon in the header to activate the camera.

🎛️ Per-Key Mapping (customizable):

Each keypad value can be assigned its own hand pose - Settings → Mapping → Hand tab → open the 9-Key/12-Key/Piano accordion for your layout. Apply a Quick Preset (like "Finger Counts") to assign every key at once, or set individual keys by hand. Which pose categories show up in those dropdowns (Poses, Pinches, Finger Counts, Advanced Shapes) is controlled in 🛠️ Developer Mode.

✋ Hand Signals (global, separate from key mapping):

  • 🤌 Chef Kiss: Clear the round.
  • 👌 OK Sign: Delete (backspace).
  • 🤘 Rock On: Play.
  • ✊ Fist: Stop.

These four work from any key mapping and fire regardless of which hand orientation (palm or knuckles facing the camera) you show. Turn on "Hand Signals" in Settings → General to enable them - it requires Hand Gestures to also be on.

💡 Tracking Rules:

  • Hold Steady: You must hold the gesture still for a fraction of a second to lock it in. This prevents accidental multi-inputs.
  • Boss Mode: Hand Tracking utilizes the camera, not the screen. It works perfectly while Boss Mode (Blackout) is active.
  • Lighting: Ensure your hand is front-lit. Heavy backlighting (like a bright window behind you) will blind the AI.
The Gesture Pad

These touch commands work on the transparent "Gesture Pad" (opened via the 🗒️ icon) or when Boss Mode Gestures are enabled. Assign any of them to a keypad value in Settings → Mapping → Touch tab; a Quick Preset can assign a whole layout at once.

💨 Basic Swipes & Taps ▼

1-Finger Swipe: Up, Down, Left, Right.

2-Finger & 3-Finger: Multi-touch swipes in any direction.

Corners: Swipe diagonally (e.g., Up-Right).

Taps: Single, Double, Triple, and Long Presses (Hold).

📐 Complex Shapes ▼

Boomerang (I-Shape): Swipe UP then DOWN without lifting.

Switchback (V-Shape): Swipe RIGHT then DOWN-LEFT without lifting.

Square/Triangle: Draw the continuous shape.

Twists: Rotate two or three fingers in a circle.

Squiggle: Rapidly scrub back and forth.

🎤 In-App Voice Input

Turn on "Voice Input" in Settings → General, then tap the 🎤 header icon to start listening. Say your Voice Trigger word (default "Set"), then a number, letter, or command in the same breath.

Examples: "Set 1 2 3" enters those values. "Set Play" plays back. Requires "Voice Commands" also on in Settings → General.

Commands: play / start / go  •  stop / pause  •  clear / reset  •  delete / backspace / undo / back  •  settings / options
