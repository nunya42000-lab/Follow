Getting Started

Welcome! Follow Me is a dual-mode memory training tool. Use the Quick Start menu to select a pre-made profile or jump right in.

Game Modes

  • Simon Says (Standard): The app generates a sequence. You repeat it. The sequence grows by one step each turn.
  • Unique Rounds: A memory test. Each round is a completely new random sequence of a specific length. You must memorize it entirely.

Controls & Gestures

  • Pinch/Expand: Use two fingers anywhere on the screen to resize the entire UI (Global Scale). You can change this to resize only the number cards in Settings.
  • Long Press Play (‚ñ∂): Toggles Autoplay mode on/off instantly.
  • Backspace (‚Üê): Removes the last entry. Hold down to speed delete.

Advanced Settings

  • Machines: In Simon mode, splits the sequence across up to 4 different "machines" (groups), making it harder to track.
  • Chunking: Breaks long sequences into smaller bites (e.g., 3 numbers, pause, 3 numbers) to help with memorization.
  • Auto Inputs: Enables the Camera and Microphone buttons. Use these to trigger inputs via sound frequency (whistling) or camera flash detection.
