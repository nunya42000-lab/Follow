Settings Glossary ‚öôÔ∏è

Game Tab
  • Profiles: Save/Load different setups (e.g., "Fast 9-Key", "Slow Piano").
  • Input: Choose between 9-Key (3x3), 12-Key (3x4), or Piano layouts.
  • Mode (Simon): Cumulative. Adds one number per round. App reads back the full history.
  • Mode (Unique): Random. New sequence every round. App forgets previous numbers.
  • Length: The amount of numbers in the sequence.
  • Machines: Split the screen for tracking up to 4 machines simultaneously.
  • Auto-Advance: (Unique Mode) Automatically clears and starts the next round after playback.
  • Practice Mode: Switches from "Tracker" to "Game". The app generates sequences for you to memorize.
Playback Tab
  • Autoplay: Immediately plays back the sequence after you enter a number.
  • Flash: Highlights buttons visually during playback.
  • Audio: Reads the numbers out loud (Text-to-Speech).
  • Morse: Vibrates a pattern for each number so you can "feel" the sequence.
  • Speed: Playback tempo (75% - 150%). Affects voice and flash speed.
  • Chunk Size: Groups numbers (e.g., "1-2-3 ... 4-5-6") primarily for multiple machines.
  • Delay: Pause between chunks. makes multiple machines easier.
  • Pause: Pause between numbers. Designed for haptic playback but works for everything
  • Voice Settings: Customize pitch rate, and volume, or choose a voice preset.
  • Haptic Mapping: Customize exactly which vibration pattern plays for each number.
General Tab
  • Timer/Counter: Adds a stopwatch or tally counter to the top header.
  • Auto Timer/Counter: Automatically starts/increments when you add the first number of a sequence. Timer stops when fully deleted.
  • Boss Mode (üåë): Turns screen black. Input Grid remains active (Top-Left = 1, Bottom-Right = Max). shake phone to start and exit.
  • BM Gestures: If enabled, Boss Mode ignores the Grid and uses Swipe gestures instead.
  • Inputs Only (ü•∑): Hold '1' to hide all buttons except the keypad.
  • Gesture Input: Press üóíÔ∏è to toggle a full-screen gesture pad.
  • AR Mode üì∏: Uses your camera as a transparent background so you can see the machine through the buttons. Toggle with the Cam icon.
  • Voice Input üé§: Toggle with the Mic icon. Say "Add One", "Plus Two" etc. Commands: "Play", "Stop", "Clear".
  • Quick Erase: Hold 'Backspace' to delete the entire sequence.
  • AP Shortcut: Long-press 'Play' to toggle Autoplay on/off.
  • Speed Gestures ‚ö°: Twist 2 fingers like a dial to adjust playback speed.
  • Vol. Gestures üîä: Twist 3 fingers like a jar lid to adjust volume.
  • Delete Gesture üßπ: Squiggle (shake finger Left-Right) to backspace.
  • Clear Gesture üí•: 2 Finger Squiggle (shake 2 fingers Left-Right) to clear everything.
  • Sequence Size: The size of the cards the numbers are displayed in.
  • UI Size: The size of everything.
  • Number Size: How big the numbers are in the cards.
  • Resize Gesture: 2 finger pinch and expand gestures to control sequence and UI size.
Mapping Tab

Configures gestures for "Gesture Input" and "BM Gestures".

  • Profiles: Presets like "Directional Swipes" (Faster) or "Standard Taps" (Easier).
  • Mapping Lists: Assign specific swipes (Up, Down, Left, Right) or taps to specific numbers.
Gesture Encyclopedia üìñ
Taps (1F / 2F / 3F) Simple touches. "2F" means two fingers touching simultaneously.
Tip: "Tap 2F" (two fingers once) is faster than "Double Tap".
Swipes (Directional) Quick slides in any of the 8 compass directions (N, NE, E, SE, S, SW, W, NW).
Long Swipe: A swipe that travels further than 150 pixels (about half the screen width).
Motion Taps (Spatial) A "lazy tap". Touch down, slide slightly (just a little bit) in a direction, and release.
Why? Allows for extremely fast inputs without needing a "clean" up/down motion.
Boomerang ü™É Swipe in a direction and immediately return to the start point without lifting.
Example: Swipe Up then Down in one continuous motion.
Shapes Draw these with one finger:
‚Ä¢ Corner: Draw an 'L' shape (any rotation).
‚Ä¢ Triangle: Draw a simple triangle.
‚Ä¢ Square: Draw a box.
(CW = Clockwise, CCW = Counter-Clockwise)
Finger Alignment For 2-finger and 3-finger taps, the app sees how you hold your hand:
‚Ä¢ Horizontal: Fingers side-by-side (like holding a burger).
‚Ä¢ Vertical: Fingers stacked (one above the other).
